# Product Context Note Template for CPO Copilot

Назначение
Эта заметка фиксирует текущий рабочий контекст продукта, чтобы copilot и команда опирались на одну и ту же картину.

Когда использовать
Используй эту заметку, если продукт, сервис, внутренний продукт, проект или объект работы уже достаточно ясны.

Когда не использовать
Если продукта или ясного объекта работы пока нет, используй Exploration Context Note.

Рекомендуемое имя итогового файла в проекте
[PROJECT NOTE] Текущий продуктовый контекст

Как заполнять
Пиши просто.
Не делай длинный отчёт.
Отделяй:
- что известно;
- что является выводом;
- что является допущением;
- чего не хватает;
- какой следующий сильный ход.
