# Product Project Passport Template for CPO Copilot

Назначение
Этот шаблон помогает собрать [PROJECT PASSPORT] Паспорт проекта для ситуации, когда продукт или объект работы уже достаточно ясны.

Когда использовать
Используй этот шаблон, если продукт, сервис, внутренний продукт, проект или объект работы уже достаточно ясны.

Когда не использовать
Если продукта или ясного объекта работы пока нет, используй Exploration Project Passport Template.

Рекомендуемое имя итогового файла в проекте
[PROJECT PASSPORT] Паспорт проекта

Как заполнять
Пиши просто.
Не делай длинный отчёт.
Отделяй:
- что известно;
- что является выводом;
- что является допущением;
- чего не хватает;
- какой следующий сильный ход.
