# Exploration Project Passport Template for CPO Copilot

Назначение
Этот шаблон помогает собрать [PROJECT PASSPORT] Паспорт проекта для ситуации, когда продукта или ясного объекта работы пока нет.

Когда использовать
Используй этот шаблон, если человек ещё только выбирает:
- что вообще стоит делать;
- для кого;
- вокруг какой проблемы;
- в каком направлении есть сильнейшая гипотеза.

Когда не использовать
Если продукт или объект работы уже достаточно ясен, используй Product Project Passport Template.

Рекомендуемое имя итогового файла в проекте
[PROJECT PASSPORT] Паспорт проекта

Как заполнять
Пиши просто.
Не делай вид, что продукт уже существует.
Отделяй:
- сигналы;
- выводы;
- допущения;
- ограничения;
- следующий маленький тест.
