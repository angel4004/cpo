# Exploration Context Note Template for CPO Copilot

Назначение
Эта заметка фиксирует текущий контекст поиска направления, когда продукта или ясного объекта работы пока нет.

Когда использовать
Используй эту заметку, если человек ещё только выбирает:
- что вообще стоит делать;
- для кого;
- вокруг какой проблемы;
- в каком направлении есть сильнейшая гипотеза.

Когда не использовать
Если продукт или объект работы уже достаточно ясен, используй Product Context Note.

Рекомендуемое имя итогового файла в проекте
[PROJECT NOTE] Текущий контекст поиска направления

Как заполнять
Пиши просто.
Не делай вид, что продукт уже существует.
Отделяй:
- сигналы;
- выводы;
- допущения;
- ограничения;
- следующий маленький тест.

## 1. Зачем сейчас ищем направление
- Кто инициатор поиска
- Почему этот вопрос возник именно сейчас
- Какой результат хотим получить от exploration

## 2. На что уже есть опора
- Какие есть навыки, компетенции или опыт
- Какие есть доступы, каналы, данные, технологии, партнёры
- Где уже есть преимущество
- Что точно не хочется делать

## 3. Какие направления уже видим
- Какие идеи, рынки, проблемы или типы продуктов уже рассматриваются
- Что в каждом направлении привлекает
- Что в каждом направлении пока слабое или туманное

## 4. Для кого это может быть
- Какие аудитории сейчас кажутся важными
- Для кого проблема может быть достаточно острой
- Что про аудитории пока остаётся догадкой

- Что здесь пока не подтверждено
